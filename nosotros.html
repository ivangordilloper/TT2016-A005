Nosotros

¿Qué es?

Esta aplicación Web es una herramienta que implementa una red social especializada de lectura para hispano hablantes que cuenta con distintas opciones, el usuario puede crear distintas listas de preferencias de libros y así establecer y mantener relaciones con personas que compartan los mismos gustos de lectura e intereses.

¿Por qué?

México es considerado un país en donde se lee muy poco a comparación de otros países, ya que aquí se leen 5.3 libros en promedio al año, según la Encuesta Nacional de Lectura 2015 (ENLE) Es por ello que en este proyecto se busca atraer a las personas al mundo de la lectura y ayudar a la comunidad lectura a tener una lectura mucho más enriquecedora. Existen distintas aplicaciones en el mercado similares a la que se desarrollará sin embargo la innovación de este proyecto es el uso de las tecnologías de la Web semántica, dando un mayor significado a la información y dotando a la aplicación la capacidad para procesar información de cada usuario de manera automática por la computadora, usando razonadores, combinándola e inferir información [8] relacionada acerca de los lectores y sus lecturas de acuerdo a los gustos y actividades en la red, para conformar grupos de usuarios con gustos similares a él y dar recomendaciones personalizadas de acuerdo a sus preferencias. Cabe mencionar que toda la información que se utiliza para hacer las listas de recomendaciones proviene tanto de la información registrada por el usuario como de la recopilada vía Facebook, esto se implementará usando la API de Facebook, que recopila todas las páginas de preferencia de determinado usuario. Otro factor novedoso será el uso de la geolocalización, con este módulo el usuario podrá obtener las ubicaciones de las librerías, como también de los usuarios que se relacionan con él. Para la correcta realización de este trabajo se requiere la aplicación de conocimientos adquiridos durante la carrera de Ingeniería en Sistemas Computacionales de la Escuela Superior de Cómputo. Las asignaturas que serán de gran utilidad para la aplicación Web son tanto de la rama de ingeniería como de sociales, entre ellas destacan las siguientes; Web Application Development, Tecnologías para la Web, Bases de Datos, Distributed Data Base, Desarrollo de Sistemas Distribuidos, Ingeniería de Software, Análisis y Diseño Orientados a Objetos, Programación Orientada a Objetos, Estructura de datos, Comunicación Oral y Escrita, Liderazgo y Administración de Proyectos.
